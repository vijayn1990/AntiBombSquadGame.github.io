Welcome to MineSweeper game
Goal of the game:
  • You need to identify all the Bombs or click on all the tiles which doesnot contain the bomb to win the game
  • Complete the game within seconds to win

    • Once game is started
  • You cannot pause the game
  • Click on tiles/buttons one after the other
  • Number 1 on tile indicate 1 bomb in its surrounding. Same logic for 2,3 and so on upto number 8
  • You lose the game when you click containing a Bomb or letter B
  • If you want to start a new game, click on Reset game button

    • Once game is lost
  • Alert box shows game is lost. You can start the game again or click Cancel to review the last game
  • All tiles with bombs will be auto-unvailed
  • Tiles with Bombs will be shown with bomb image
  • Tiles with rightly marked bombs as shown in Black bomb image
  • Incorrect tiles marked/flagged as a bomb, but not containing a bomb will be shown with X with Black background

    • Once game is won
  • Alert box shows game is won
  • All tiles with bombs will be auto-unvailed
  • Tiles with Bombs will be shown with bomb image

    • Note: Left click will open the tile. Right click will flag as a suspect tile to contain bomb.

    Seconds left: 300 Bombs to be identified: 10
